## 📘 About this repository

This repository is an interactive learning project that teaches the foundations of GitHub:

- Creating and switching branches
- Making commits
- Opening pull requests
- Merging changes back into `main`

It is designed to be self-paced and beginner-friendly.

## 🎯 Learning path

The project is organized as a guided 4-step flow:

1. **Create a branch**
2. **Commit a file**
3. **Open a pull request**
4. **Merge your pull request**

When each step is completed, GitHub Actions workflows validate progress and unlock the next stage.

## 🗂️ Repository structure

| Path | Purpose |
|---|---|
| `.github/steps/` | Step-by-step lesson content and instructions |
| `.github/workflows/` | Automation that checks exercise progress |
| `.github/images/` | Visual guides used in lesson instructions |
| `README.md` | Main overview for the repository |
| `LICENSE` | MIT license information |

## 🚀 How to use this project

1. Open the exercise issue from the button above.
2. Follow each step in order.
3. Complete actions directly in GitHub (branch, commit, PR, merge).
4. Wait for automated feedback after each step.

## ✅ Skills you will practice

- Basic GitHub navigation
- Branch-based development workflow
- Clear commit messaging
- Pull request collaboration
- Safe merge process
